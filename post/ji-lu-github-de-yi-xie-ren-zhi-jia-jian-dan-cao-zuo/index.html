--------手动分界线QAQ

一.github

A：官网意思就是为开发者提供git仓库的托管服务，让自己和他人共享代码，即远隔半个地球也可以可以共同开发软件。

图标是octocat（章鱼猫）如图
这里还有 一些网友自己制造的章鱼猫的形象 的链接（QAQ只是想粘来玩玩嘻嘻嘻https://octodex.github.com/%EF%BC%8C

B：正题：区分好github和git，两个绝对不是一个东西！不是缩写啥子的，git可以说是一个仓库，就存放那些开发者的代码，而github就相当于取用仓库内东西的地方，那也就是说，github上取用东西的所有的源代码都是git管理的.

二.github的创建和准备
A.先打开github的创建页面（要加链接，我是刚开始一直打不开（外国网站），然后就用手机去注册了，苹果端直接在sari上搜就好，安卓的下载火狐浏览器，（其他浏览器没试，不过安卓系统自带的浏览器也不行。）在上面可以登进github完成注册（亲测有效），注册的流程巴拉巴拉，可能明天更吧..

三.Git

A：官网意思就是：分散型版本管理系统，，简单的说就是跟日记一样，记录每次更新的记录，就让自己方便看到自己添加或者删改代码的过程，假若删错了也可以找回被删除的文件。

B：记录一下git的安装吧 https://git-scm.com/download/win 这里有安装地址，复制粘贴到网页上选择自己的机型就可以下载啦，下载好了之后会出现类似于命令行的框框，能输入的地方引入$
1.先在上面输入 $ git config --global user.name(左边一堆都是固定格式)，"Radiantzy"
即 $ git config --global user.name "Radiantzy"
2.再在上面输入 $ git config --global user.email "1660098139@qq.com"
